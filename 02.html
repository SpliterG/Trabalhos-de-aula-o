Diferenças de CSS Inline, interno e externo.

  1. Inline

    2. Estilos inline, são utilizados para aplicar regras de estilo únicas à um elemento, colocando as regras do CSS diretamente na tag inicial. Pode ser anexado a um elemento usando o atributo <.style>.

  2. Interno

    3. Estilos internos, só afetam o documento onde estarão sendo incorporadas.

  3. Externo

    4. Estilos externos é ideal quando o estilo é aplicado em muitas paginas do site. Estilo externo contem todas as regras de estilo em um documento separado que você pode vincular de qualquer arquivo HTML em seu site.

O que é um seletor:

Um seletor CSS é a primeira parte de uma regra CSS. É um padrão de elementos e outros termos que informam ao navegador quais elementos HTML devem ser selecionados para que os valores de propriedade CSS dentro da regra sejam aplicados a eles.

  1. Classe

    2. O seletor classe é usado para selecionar todos os elementos que pertencem a um atributo de classe específico.

    sintaxe:.nome da classe

5 propriedades css.

  1. Background-color>Altera a cor de fundo de um elemento.

  2. Font-size>Você pode mudar o tamanho de sua fonte.

  3. Margin-left:Cria uma margem à esquerda.

  4. Margin-Right:Cria uma margem à direita.

  5. Border-right:Cria uma borda à direita

  6. Border-left:Cria uma borda à esquerda.
